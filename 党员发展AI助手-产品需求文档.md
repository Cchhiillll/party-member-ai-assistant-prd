# 党员发展AI助手 - 产品需求文档 (PRD)

## 一、产品定位

面向党务工作者与基层党组织的**党员发展AI助手**，通过 AI 技术提升党员发展全流程效率，降低党务工作者工作负担，提供智能化办事引导与内容生成能力。

### 核心价值

| 价值点 | 说明 |
|-------|------|
| **效率提升** | 党员活动线上准备时间减少 50% |
| **可控生成** | AI 内容有来源、可复核 |
| **智能引导** | 更智能化、更便捷的办事引导 |

---

| 模块 | 功能 | 优先级 |
|-----|------|-------|
| **PC端AI助手** | 办事引导、智能问答 | P0 |

### 2.2 详细功能说明

#### 功能1：PC端AI助手（办事引导）

**功能描述**
- 为党务工作者和申请人提供智能办事引导
- 回答党员发展流程相关问题
- 指引用户完成各项操作

**功能要点**

| 功能点 | 描述 |
|-------|------|
| 流程查询 | 回答"我现在该做什么""下一步是什么"等问题 |
| 政策解答 | 解答入党条件、材料要求等政策问题 |
| 操作指引 | 指引用户找到正确的办事入口 |
| 材料清单 | 自动生成当前阶段所需材料清单 |

**交互设计**

```
用户: 我刚提交了入党申请书，接下来要做什么？

AI助手: 您好！提交入党申请书后，接下来的流程是：
1. 等待党办接口人审批您的申请书（预计3-5个工作日）
2. 审批通过后，支部接口人会与您进行入党谈话
3. 谈话结束后，支部接口人会录入谈话记录
4. 后续会通知您是否进入积极分子培养阶段

💡 提醒：您可以随时在这里询问流程进展
```

**验收标准**

| 指标 | 标准 |
|-----|------|
| 问题准确率 | ≥ 90% |
| 响应时间 | ≤ 3秒 |
| 用户满意度 | ≥ 4.0/5.0 |
| 覆盖场景 | 覆盖100%发展流程节点 |

#### 功能2：入党申请书AI内容审核及校验

**功能描述**
- 自动审核入党申请书内容完整性
- 校验格式规范性
- 识别敏感/违规内容
- 生成审核意见与修改建议

**功能要点**

| 功能点 | 描述 |
|-------|------|
| 完整性检查 | 检查必填内容是否完整（个人情况、入党动机、对党的认识等） |
| 格式校验 | 检查格式是否规范（标题、称呼、正文、落款等） |
| 内容审核 | 识别政治敏感内容、逻辑问题、表述不当等 |
| 相似度检测 | 检测是否存在抄袭/模板化痕迹 |
| 修改建议 | 生成具体修改建议 |

**审核维度**

| 维度 | 检查项 | 权重 |
|-----|-------|------|
| **完整性** | 入党动机、个人经历、对党的认识、表态内容 | 30% |
| **规范性** | 格式、字数、段落结构 | 20% |
| **真实性** | 内容真实性、个人特色、避免模板化 | 25% |
| **正确性** | 政治立场、表述准确、无敏感内容 | 25% |

**输出示例**

```
📋 审核结果：需修改

✅ 通过项：
- 格式规范，包含标题、正文、落款
- 入党动机表述清晰

⚠️ 待改进：
1. 【完整性】缺少对个人成长经历的描述
   建议：补充学习、工作经历及思想转变过程
   
2. 【真实性】内容与模板相似度较高(68%)
   建议：增加个人真实经历和感悟，体现个人特色
   
3. 【正确性】第二段"三个代表"表述不完整
   建议：修正为完整表述

综合评分：72/100
建议：修改后重新提交
```

**验收标准**

| 指标 | 标准 |
|-----|------|
| 审核准确率 | ≥ 85% |
| 误报率 | ≤ 10% |
| 漏报率 | ≤ 5% |
| 审核速度 | ≤ 10秒/份 |
| 修改建议有效性 | ≥ 80%用户认可 |

## 三、目标用户

### 3.1 用户画像

| 用户类型 | 角色 | 核心需求 | 痛点 |
|---------|------|---------|------|
| **党务管理员** | 支部书记/组织委员 | 高效管理流程、减少重复工作 | 审核效率低、重复填写 |
| **发展对象** | 群众/积极分子/预备党员 | 清晰指引、便捷办事 | 流程不熟、找不到入口 |
| **督导人员** | 上级督导/纪检 | 监督检查、数据查阅 | （暂未覆盖） |

### 3.2 用户旅程

**发展对象视角**

```
提交申请书 → 等待审核 → 参加谈话 → 确认积极分子 → 
培养考察 → 确认发展对象 → 政审 → 确认预备党员 → 
预备期考察 → 确认正式党员 → 归档
```

**党务管理员视角**

```
审核申请书 → 安排谈话 → 录入记录 → 组织支委会 → 
填报信息 → 组织党员大会 → 建档 → 归档
```

## 四、发展流程

### 4.1 完整流程图

```
┌─────────────────────────────────────────────────────────────┐
│                      党员发展全流程                           │
├─────────────────────────────────────────────────────────────┤
│  入党申请 → 积极分子 → 发展对象 → 预备党员 → 正式党员 → 归档  │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 各子流程详解

#### 4.2.1 入党申请流程

```
开始 
  → 【线上】申请人发起申请流程
  → 【线上】申请人提交申请书
  → 【线上】党办接口人审批申请书
  → 【线下】支部接口人与申请人谈话
  → 【线上】支部接口人录入谈话记录
  → 结束
```

#### 4.2.2 确认入党积极分子流程

```
开始
  → 【线上】党办发起流程
  → 【线下】召开支委会，确认积极分子，指定培养人
  → 【线上】支部接口人审批通过并线上指定培养人
  → 结束
```

#### 4.2.3 确认发展对象流程

```
开始
  → 【线上】党办发起流程
  → 【线下】召开支委会，确认发展对象
  → 【线上】支部接口人填写支委会时间及形式并审批通过
  → 结束
```

#### 4.2.4 确认预备党员流程

```
开始
  → 【线上】党办发起流程
  → 【线下】召开支部党员大会，确认预备党员
  → 【线上】支部接口人填写支部党员大会时间及形式并审批通过
  → 【线上】党办接口人填写最终确认的预备党员时间并审批通过
  → 结束
```

#### 4.2.5 建档流程

```
开始
  → 【线上】党办发起流程
  → 【线上】党办接口人基于前述流程信息，录入入党志愿书编号及附件
  → 审批通过
  → 结束
```

#### 4.2.6 确认正式党员流程

```
开始
  → 【线上】党办发起流程
  → 【线下】召开支部党员大会，确认正式党员
  → 【线上】支部接口人填写支部党员大会时间及形式并审批通过
  → 【线上】党办接口人填写最终确认的转正时间并审批通过
  → 结束
```

#### 4.2.7 归档流程

```
开始
  → 【线上】党办发起流程
  → 【线上】党办接口人核对党员信息、党籍信息、入党志愿书及附件
  → 审批通过
  → 结束
```

### 4.3 接续流程

#### 4.3.1 预备党员接续发展

```
转入流程完成 → 党办手动触发接续任务 → 进入建档流程子流程
```

#### 4.3.2 发展党员接续发展

```
党办发起发展党员接续 → 进入确认发展党员流程
```
