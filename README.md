# Party Member AI Assistant

## 党员发展AI助手

面向党务工作者与基层党组织的智能助手

## 功能模块

- PC端AI助手（办事引导）
- 入党申请书AI审核
- 谈话记录结构化生成
- 定制化消息提醒
- 活动管理辅助

## 技术栈

- Frontend: React + TypeScript + Ant Design
- Backend: Node.js + Express + TypeScript
- AI: OpenAI / Claude / GLM API

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 构建
npm run build
```

## 项目结构

```
party-member-ai-assistant/
├── frontend/          # 前端项目
├── backend/           # 后端项目
├── docs/              # 文档
└── package.json       # 根配置
```
